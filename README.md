# Bootflow – Shop Assist for WooCommerce (Free Version)

A lightweight, local-first WooCommerce shop assistant that helps customers find products faster through smart search, comparison, contact shortcuts, delivery information, and browser-based voice input.

This free version is focused on practical storefront product discovery without forcing external AI services for the core experience.

## Free Version Features

- Smart product search using keyword and fuzzy matching
- Product cards with image fallback handling
- Side-by-side product comparison
- Built-in browser voice input where supported
- Custom keyword-based responses for store FAQs and policies
- Starter questions configurable from the admin panel
- Delivery and contact information from store configuration
- White-label options for chatbot name, icon, and welcome text
- Multiple built-in languages
- Local-first core flow without mandatory remote providers

## What This Plugin Does

Bootflow Shop Assist adds a floating WooCommerce chatbot widget to your storefront.
Customers can search for products in natural language, compare matching items, open product details, and get quick store information such as delivery and contact details.

Unlike overly complex chatbot tools, the free version is designed to stay predictable and useful for real WooCommerce stores.
Its main job is product discovery, not pretending to answer every possible question with remote AI.

## Installation

### From WordPress.org

1. Go to Plugins -> Add New in your WordPress admin.
2. Search for Bootflow Shop Assist for WooCommerce.
3. Click Install Now and then Activate.
4. Open the Shop Assist settings page in wp-admin.

### Manual Installation

1. Download the plugin ZIP.
2. Upload it via Plugins -> Add New -> Upload Plugin.
3. Activate the plugin.
4. Configure chatbot text, colors, starter questions, and contact settings.

## Requirements

- WordPress 5.8 or higher
- WooCommerce 5.0 or higher
- PHP 7.4 or higher

## How It Works

1. The plugin exports local store product data for fast matching.
2. Customers type or speak a query in the floating chatbot.
3. The plugin searches WooCommerce products using local keyword and fuzzy logic.
4. Matching products are shown as cards with actions like details, compare, and add to cart.

## Use Cases

- Help visitors find products faster than browsing category pages
- Surface popular, sale, or lower-priced products through quick queries
- Give customers a simpler mobile-friendly product search experience
- Provide quick delivery and contact information inside the chat widget
- Add lightweight storefront assistance without depending on external AI APIs

## Privacy and Data Handling

- Core product search runs locally on the WordPress site
- No mandatory external AI provider is required for the free plugin flow
- No embedded analytics scripts are required for the free version
- Chat UI can display a privacy/GDPR notice configured by the site owner
